Description

Arcus is a browser based, user interface into the powerful Dispatch system. Arcus is ideal for distributed call center operations, and allows primary functions including call taking and dispatching, vehicle tracking and account/driver/vehicle administration.

Optional features include basic cashiering, SMS text notifications to passengers, as well as automated or driver initiated call outs.

Arcus Administration was developed as a part of the Arcus Enterprise Project, but it also can be used separately. Arcus Administration allows authorized personnel to manage operator, driver and vehicle profiles. Configurable parameters allow you to tailor Arcus to your business processes.
